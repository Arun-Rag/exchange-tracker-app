# Exchange Tracker App

This project is a web-based application for managing and tracking exchange processes.

## Features
- Track exchanges using JSON data
- Generate documents using JavaScript scripts
- Support for import/export data (Excel, JSON)
- Multiple app versions (Shared, Prototype)

## Technologies Used
- HTML
- JavaScript
- Python
- JSON

## Files
- Exchange_Tracker_App.html → Main application
- exchanges.json → Data storage
- gen_doc.js → Document generation
- build_app.py → Build script

## How to Run

1. Open `Exchange_Tracker_App.html` in browser
2. Load or edit JSON data
3. Use scripts to generate outputs
